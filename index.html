Privacy Policy

LineLib — Software Commands Reference · Last updated: February 18, 2026

Introduction

LineLib ("we", "our", or "the app") is a software commands reference application. We are committed to protecting your privacy. This Privacy Policy explains how we handle information when you use our application.

Information We Collect

We do not collect any personal data. LineLib does not require an account, login, or any personal information to function.

Data Storage

All data created within the app (such as saved commands, collections, and preferences) is stored locally on your device only. This data is never transmitted to any server or third party.

  • Saved commands and collections — stored on device
  • App preferences and settings — stored on device
  • Search history — not stored

AI Assistant Feature

LineLib includes an optional AI Assistant feature that helps users with software-related questions. This feature requires an internet connection.

When you use the AI Assistant:

  • Your questions are sent to OpenRouter (a third-party AI API service) for processing.
  • Only the text of your question is transmitted — no personal data, device information, or usage analytics are included.
  • The AI is restricted to answering questions about 3D modeling and 2D design software only.
  • Questions and responses are not stored on any server. They exist only in memory during your app session.
  • OpenRouter's privacy policy can be found at: openrouter.ai/privacy

The AI Assistant has a daily usage limit and does not process questions unrelated to software commands and shortcuts.

Third-Party Services

LineLib uses the following third-party service:

  • OpenRouter API — Used exclusively for the AI Assistant feature. No personal data is shared. Only the text of user questions about software commands is transmitted.

LineLib does not integrate with any analytics, advertising, or tracking services.

Internet Access

LineLib's core functionality (browsing commands and shortcuts) works entirely offline. Internet access is only required for the optional AI Assistant feature.

Children's Privacy

LineLib does not collect any personal information from anyone, including children under the age of 13. The app is safe for users of all ages.

Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be reflected on this page with an updated revision date.
